# Surya Labs Workspace — plugin exemplo (opcional)

**Não é parte do core.** Cópia de referência do pacote Foundation. O canônico operacional vive no repo `surya-labs-workspace` em `integrations/sky-forge/`.

## Uso

1. Exportar pacote: `sky export` com profile `consulting-handoff`
2. Consumidor importa pasta do pacote via adapter (fora deste repo)

## Arquivos

- `plugin.yaml` — declaração do adapter
- `mappings/` — mapeamentos file-contract
- `templates/` — tradução pt-BR (proposta, handoff, gates)

## Compatibilidade

Profile: `consulting-handoff` apenas (MVP).
