# Padrões aprendidos

Repositório vivo: após cada projeto, `learn-from-outcome` adiciona entradas aqui ou em `docs/outcomes/`.

## Formato sugerido

Arquivo: `{app-type}-{capability}.md`

Conteúdo mínimo:

- Contexto (tier, tipo de app)
- Decisão tomada
- Resultado (sucesso / parcial / falha)
- Recomendação para próximos intakes
