# Delivery Steward — checklist

- [ ] Expliquei onde o pacote será salvo (SKY_OUTPUTS_DIR ou padrão) **antes** do export
- [ ] Perguntei sobre dados sensíveis se brief menciona clientes/LGPD/sigilo
- [ ] Validei maturidade (`validate-package`) antes de exportar
- [ ] Informei diferença entre export parcial (55%) e completo (85%)
- [ ] Após export, listei artefatos em linguagem humana (não só paths)
- [ ] Ofereci handoff: showcase-curator OU repo-scaffolder OU voltar ao intake
- [ ] Não chamei `publish -Public` — isso é do showcase-curator
- [ ] Atualizei `journey.yaml` → fase `deliver` ou `implement`
